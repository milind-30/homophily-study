# Data Folder

This `data/` folder holds notebooks and scripts for actively processing data, plus subfolders:
- **csv/**: Current or actively used CSVs (e.g., `prolific_first_pass_data.csv`).
- **jsonl/**: JSON log files from the most recent Prolific runs.
- **helpers.py**: Utility functions for data wrangling.
- **convert_pairwise.ipynb**, etc.: Notebooks for converting raw data into analysis-ready formats.

Older or unused data has been moved to `archive/data`.
